## 简介

框架支持redis或者内存
在`application-dev/prod.yml`里可以切换
```yml
# 缓存使用类型
cache:
  # memory为使用内存，redis为使用Redis
  type: memory

```
## 如果要用redis
1. cache.type 改为 redis
2. pom里取消注释sa-token集成redis
    ```xml
    <!-- Sa-Token 整合 Redis （使用 jackson 序列化方式） -->
    <dependency>
        <groupId>cn.dev33</groupId>
        <artifactId>sa-token-redis-jackson</artifactId>
        <version>${sa-token.version}</version>
    </dependency>
    ```
3. 在`application-dev/prod.yml`的`spring.data.redis`节点下配置redis连接信息

## 如果要用内存
1. cache.type 改为 memory
2. pom里注释sa-token集成redis
    ```xml
    <!-- Sa-Token 整合 Redis （使用 jackson 序列化方式） -->
    <dependency>
        <groupId>cn.dev33</groupId>
        <artifactId>sa-token-redis-jackson</artifactId>
        <version>${sa-token.version}</version>
    </dependency>
    ```
3. 在`application-dev/prod.yml`的`spring.data.redis`节点可以注释掉或者删掉

## 使用方式

在需要使用的地方，直接使用工具类`CacheUtil`，不管你是用的redis还是内存，都是通过该工具类来操作的
