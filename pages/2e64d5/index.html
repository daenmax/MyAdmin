防重提交注解

# 说明

防重提交用于防止某些情况下，在短时间内提交多次一样的参数

框架增加了一个注解@RepeatSubmit

有一个参数

interval：每次提交最少间隔的时间数，单位为下一个参数，默认1000 timeUnit：时间单位，默认毫秒 message：提示消息，默认 请勿重复提交

# 使用例程

@GetMapping("/test2")
@RepeatSubmit
public Result test2() {
  return Result.ok("查询成功");
}
